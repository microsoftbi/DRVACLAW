# DRVACLAW 驾驶陪练系统技能

## 功能

该技能提供对接驾驶陪练系统的接口，支持以下功能：

1. **创建预约课程**：调用 `create_appointment` 方法创建预约课程
2. **创建充值记录**：调用 `create_recharge` 方法创建充值记录
3. **查询课程状态**：调用 `get_confirmed_courses_by_date` 方法查询指定日期的已确认课程

## 触发方式

当用户说"老赵"时触发该技能。

## 意图识别

技能会判断用户的意图，并执行相应的操作：

### 1. 创建预约课程

**触发条件**：当用户的意图是创建一个课程。

**需要确认的信息**：
1. 学员名
2. 学员区域
3. 教练名
4. 教练区域
5. 日期（格式：YYYY-MM-DD）
6. 开始时间（格式：1-24）
7. 结束时间（格式：1-24）

当这些信息都确认之后，让用户确认这些信息，然后再去调用方法。

**示例**：
- 用户："老赵，帮我创建一个课程，学员是张三，区域是中原区，教练是李四，区域是中原区，日期是2026-04-15，开始时间是10:00，结束时间是12:00。"
- 系统：确认信息后创建预约课程

### 2. 创建充值

**触发条件**：当用户的意图是充值。

**需要确认的信息**：
1. 学员手机号
2. 充值金额
3. 获得课程数

当这些信息都确认之后，让用户确认这些信息，然后再去调用方法。

**示例**：
- 用户："老赵，帮我给学员13312341234充值1000元10节课。"
- 用户："老赵，帮我给学员13312341234充值。"，系统会问用户的充值金额是多少，然后问充的课程是多少节。

### 3. 查询课程状态

**触发条件**：当用户的意图是查询某天的课程。

**需要确认的信息**：
1. 日期（格式：YYYY-MM-DD）

**示例**：
- 用户："老赵，帮我查询2026-04-15的课程状态。"
- 系统：返回这一天所有已确认的课程列表

## 实现细节

该技能会调用 `SKILLS/SKILL.py` 中的方法：

1. **create_appointment**：创建预约课程
2. **create_recharge**：创建充值记录
3. **get_confirmed_courses_by_date**：查询指定日期的已确认课程

## 依赖项

- `requests`：用于发送HTTP请求
- `datetime`：用于处理日期时间

## 安装依赖

```bash
pip install requests
```

## 示例调用

### 创建预约课程
```python
result = create_appointment(
    student_name="张三",
    student_area="中原区",
    coach_name="李四",
    coach_area="中原区",
    date="2026-04-15",
    start_time=10,
    end_time=12
)
print(result)
```

### 创建充值记录
```python
result = create_recharge(
    student_phone="13312341234",
    amount=1000,
    courses=10
)
print(result)
```

### 查询课程状态
```python
result = get_confirmed_courses_by_date("2026-04-15")
print(result)
```
