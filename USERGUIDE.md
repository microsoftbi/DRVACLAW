# 驾驶陪练小龙虾管理系统 - 用户使用手册

## 系统简介

驾驶陪练小龙虾管理系统是一款专为驾驶陪练机构设计的管理软件，旨在帮助机构高效管理学员、教练、预约、充值等业务流程，提高管理效率和服务质量。

## 系统功能

系统主要包含以下功能模块：

1. **预约管理**：管理学员的课程预约，包括创建、确认、取消、删除预约等操作。
2. **充值管理**：管理学员的充值记录，包括创建、修改、删除充值记录等操作。
3. **余额查询**：查询学员的余额记录，包括充值和课时消耗记录。
4. **区域管理**：管理训练区域信息，包括创建、修改、删除区域等操作。
5. **人员管理**：管理学员和教练信息，包括创建、修改、删除人员信息等操作。
6. **区域统计**：统计各区域的预约情况，包括预约数量、完成情况等。
7. **操作审计**：记录系统中的所有重要操作，包括预约、充值等操作的审计记录。

## 系统安装与启动

### 后端启动
1. 进入后端目录：`cd Backend`
2. 激活虚拟环境：`source .venv/bin/activate`（Mac/Linux）或 `.venv\Scripts\activate`（Windows）
3. 启动后端服务：`python main.py`
4. 后端服务默认运行在 `http://localhost:8002`

### 前端启动
1. 进入前端目录：`cd Frontend`
2. 安装依赖：`npm install`
3. 启动前端服务：`npm run dev`
4. 前端服务默认运行在 `http://localhost:5173`

## 功能操作指南

### 1. 预约管理

**功能说明**：管理学员的课程预约，包括创建、确认、取消、删除预约等操作。

**操作步骤**：
1. 点击左侧菜单中的「预约管理」进入预约管理页面。
2. **创建预约**：点击「新增预约」按钮，填写预约信息（学员、教练、区域、日期、开始时间、结束时间、状态），点击「保存」按钮。
3. **确认预约**：在预约列表中，找到状态为「待确认」的预约，点击「确认」按钮。
4. **取消预约**：在预约列表中，找到状态为「已确认」的预约，点击「取消」按钮。
5. **删除预约**：在预约列表中，找到要删除的预约，点击「删除」按钮。
6. **筛选预约**：使用页面顶部的筛选器，按学员、教练、区域、状态等条件筛选预约。

**注意事项**：
- 预约时间必须是未来一周的日期。
- 开始时间和结束时间必须是整数（如9表示9:00）。
- 当预约状态被标记为「已完成」时，系统会自动在余额表中生成一条课时消耗记录。

### 2. 充值管理

**功能说明**：管理学员的充值记录，包括创建、修改、删除充值记录等操作。

**操作步骤**：
1. 点击左侧菜单中的「充值管理」进入充值管理页面。
2. **创建充值**：点击「新增充值」按钮，填写充值信息（学员、金额、课时数、充值时间），点击「保存」按钮。
3. **修改充值**：在充值列表中，找到要修改的充值记录，点击「编辑」按钮，修改信息后点击「保存」按钮。
4. **删除充值**：在充值列表中，找到要删除的充值记录，点击「删除」按钮。
5. **筛选充值**：使用页面顶部的筛选器，按学员、日期等条件筛选充值记录。

**注意事项**：
- 充值时间使用北京时间。
- 当创建充值记录时，系统会自动在余额表中生成一条充值记录。

### 3. 余额查询

**功能说明**：查询学员的余额记录，包括充值和课时消耗记录。

**操作步骤**：
1. 点击左侧菜单中的「余额查询」进入余额查询页面。
2. **筛选余额记录**：使用页面顶部的筛选器，按学员、区域、类型等条件筛选余额记录。
3. **查看充值详情**：在余额记录列表中，找到类型为「充值」的记录，点击「查看详情」按钮，弹出充值详情子窗体。
4. **查看预约详情**：在余额记录列表中，找到类型为「课时消耗」的记录，点击「查看详情」按钮，弹出预约详情子窗体。

**注意事项**：
- 余额记录按日期从早到晚排列。
- 课时消耗记录的课时值显示为负数。
- 系统会自动计算每个学员的结余信息。

### 4. 区域管理

**功能说明**：管理训练区域信息，包括创建、修改、删除区域等操作。

**操作步骤**：
1. 点击左侧菜单中的「区域管理」进入区域管理页面。
2. **创建区域**：点击「新增区域」按钮，填写区域名称，点击「保存」按钮。
3. **修改区域**：在区域列表中，找到要修改的区域，点击「编辑」按钮，修改名称后点击「保存」按钮。
4. **删除区域**：在区域列表中，找到要删除的区域，点击「删除」按钮。

### 5. 人员管理

**功能说明**：管理学员和教练信息，包括创建、修改、删除人员信息等操作。

**操作步骤**：
1. 点击左侧菜单中的「人员管理」进入人员管理页面。
2. **创建人员**：点击「新增人员」按钮，填写人员信息（姓名、类型、区域），点击「保存」按钮。
3. **修改人员**：在人员列表中，找到要修改的人员，点击「编辑」按钮，修改信息后点击「保存」按钮。
4. **删除人员**：在人员列表中，找到要删除的人员，点击「删除」按钮。
5. **筛选人员**：使用页面顶部的筛选器，按类型、区域等条件筛选人员。

### 6. 区域统计

**功能说明**：统计各区域的预约情况，包括预约数量、完成情况等。

**操作步骤**：
1. 点击左侧菜单中的「区域统计」进入区域统计页面。
2. 系统会自动统计各区域的预约数量、已完成数量、未完成数量等信息。
3. 可以查看每个区域的详细预约情况。

### 7. 操作审计

**功能说明**：记录系统中的所有重要操作，包括预约、充值等操作的审计记录。

**操作步骤**：
1. 点击左侧菜单中的「操作审计」进入操作审计页面。
2. **筛选审计记录**：使用页面顶部的筛选器，按学员、教练等条件筛选审计记录。
3. 查看审计记录列表，包括操作人、操作时间、操作类型、操作结果等信息。

**注意事项**：
- 审计记录包含操作的详细信息，如预约ID、充值ID等。
- 审计记录按操作时间倒序排列。
