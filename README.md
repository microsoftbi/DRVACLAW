# 驾驶陪练小龙虾

## 简介
这个是我Vibe Coding练手的一个项目。
不同于先前的另外一个项目，那个项目强调多角色，这个项目只关注管理员角色。
项目的来源是我的驾驶陪驾教练，苦于一直没有一个系统来维护他们的业务信息。
于是我根据他们业务比较浅显的理解，简单搭建了这个系统。

## 功能
驾驶陪练管理系统，为刚拿到驾驶证还不敢上路，以及长时间没开车需要重新熟悉驾驶的司机。
学员联系平台管理员，说明地点和时间，然后由管理员指派教练。
这个系统只有一个管理员角色，学员和教练的信息都靠管理员来维护。
然后系统支持对接小龙虾，用户可以在微信里下各种指令来操作系统，这个系统的技术尝试主要是为了这个。

## Vibe Coding平台

Doubao-Seed 2.0 code
DBeaver CE

## 技术架构
- Vue3
- FastAPI(Python3.11)
- DAO（没有使用SQLAlchemy）
- SQLLite
